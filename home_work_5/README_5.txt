Война и мир_книга.txt - книга для работы и выполнения задания

TextRead - класс с единственным методом readTextFile() - читает текстовый файл
 и конвертирует текст из данного текстового файла в строку;

п. 2.1 - 2.2 решается в классе TextCollection методами:

formatTextToWords(String text) - форматирует строку и разбивает ее на слова, помещая в массив;

collectUniqueWords() - подсчитывает количество уникальных слов в тексте;

collectAllKeyWords() - каждое полученное слово отформатированной строки помещает
как ключ в Map, а количество повторений в тексте ставит как значение;

countTopWords(Map<String, Integer> mapWarPeace, int top) - сортирует коллекцию Map
 и выводит заданное количество (top) слов и их повторений;

пакет search содержит все классы и интерфейс, связанный с реализацией поиска слов в тексте

класс EasySearch решает п. 4.1 методом:
search(String text, String word) -метод возвращает информации о количестве раз, которое встречается слово в тексте
поиск реализован с использованием метода indexOf из класса String

класс RegExSearch решает п. 4.2 методом
search(String text, String word) -метод возвращает информации о количестве раз, которое встречается слово в тексте
поиск реализован с использованием метода indexOf из класса String

классы SearchEnginePunctuationNormalizer и SearchEnginePunctuationNormalizerDecorator решают п. 4.3

п.3  - интерфейс ISearchEngine

TextMain - результаты работы методов, использованных в п.2, 4, 5

пакет tests тестирует работу созданных реализаций интерфейса ISearchEngine

класс WorkWithDirectory решает п. 7 методами:

containFiles (File file) - метод создает список всех файлов, содержащихся в директории;

returnRightFileName(List<String> fileBooks, String fileName) - метод, который по кусочку слова из названия файла
возвращает полный адрес файла

choosenMatches(String file, String word) - метод, который считает количество совпадений слова в тексте;

writeToResultFile - метод, который записывает результаты поиска пользователя в файл;

chooseWhatToDo(List<String> fileBooks) - метод, или скорее алгоритм, который позволил разгрузить мэйн,
перенеся всю ветку общения с пользователем в "черный ящик"

класс ChooseBookMain - результаты работы методов, использованных в классе WorkWithDirectory
